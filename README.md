# Fluxcore

![Fluxcore logo](.github/fluxcore-logo.svg)

Fluxcore is a rebrand of Firefli (a maintained fork of the open-source staff management platform Orbit) for managing Roblox group staff and workflows.

## Branding

Set these environment variables to fully customize links and labels:

- `NEXT_PUBLIC_BRAND_NAME` (default: `Fluxcore`)
- `NEXT_PUBLIC_BRAND_LOGO_PATH` (default: `/fluxcore-logo.svg`)
- `NEXT_PUBLIC_BRAND_DOCS_URL` (optional)
- `NEXT_PUBLIC_BRAND_GITHUB_URL` (optional)
- `NEXT_PUBLIC_BRAND_FEEDBACK_URL` (optional)
- `NEXT_PUBLIC_BRAND_COFFEE_URL` (optional)
- `NEXT_PUBLIC_BRAND_COPYRIGHT_NAME` (default: `Fluxcore`)
